# Guardrails Scanner Runtime

The proprietary extension-security runtime used by Guardrails CLI and
Guardrails Deep Scan.

## Guardrails CLI

Install the local extension scanner:

```bash
pipx install guardlens
```

Run it with:

```bash
guardrails
```

Guardrails CLI scans extensions installed in supported VS Code-compatible
editors without executing extension code.

Website: [ide-scanner.vercel.app](https://ide-scanner.vercel.app)
